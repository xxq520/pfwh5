# wholesaleWeb

批发网前端

> 批发网主要有两种用户
> > 供应链用户
> >
> > 批发网用户

供应链代码路径：/src/pages/block

供应链访问路径：/block/***

我们的技术栈基于 ES2015+、vue、vuex、vue-router，提前了解和学习这些知识会非常有帮助

Install `node_modules`:

```bash
npm install
```
or

```bash
yarn
```

### Start project

```bash
npm run dev
```

### Build project

```bash
npm run build
```

## 项目结构
```
   ├── config                   # 构建等配置
   ├── src
   │   ├── assets               # 本地静态资源
   │   ├── components           # 业务通用组件
   │   ├── pages                # 业务页面入口和常用模板
   │   ├── router               # 路由配置文件
   │   ├── store                # 全局状态管理模块 vuex
   ├── README.md
   └── package.json

```
